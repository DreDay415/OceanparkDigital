
# Ocean Park Digital v2 (Next.js Cinematic Edition)

This is the upgraded, high-performance, cinematic version of the Ocean Park Digital website.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion + Lenis Smooth Scroll
- **3D**: React Three Fiber + Drei (Ocean Simulation)
- **Icons**: Lucide React
- **Fonts**: Inter & Playfair Display

## Getting Started

1. Navigate to this directory:
   ```bash
   cd ocean-park-next
   ```

2. Install dependencies (if you haven't):
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)

## Key Features
- **Cinematic Ocean Hero**: Real-time 3D ocean simulation using WebGL.
- **Glassmorphism Design**: Modern, premium aesthetic with blur effects.
- **Smooth Scrolling**: "Heavy" premium feel using Lenis.
- **Dynamic Blog**: Fully functional blog with static generation for SEO.
- **Performance**: Optimized for Core Web Vitals.
